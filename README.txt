Project Name: 
Customer Churn Prediction System
Domain: 
Machine Learning

Description:
This project analyzes customer behavior patterns to predict churn for TEYZIX CORE. It includes a complete Machine Learning pipeline featuring Exploratory Data Analysis, Feature Engineering, and model training. An Optimized Random Forest model was selected as the primary prediction engine, achieving ~80% accuracy. The project also includes SHAP explainability to highlight top churn drivers and a comprehensive Business Insights Report.

Files Included:

Task-1_ ML-INT-1 .ipynb (Main Python Code)

WA_Fn-UseC_-Telco-Customer-Churn.csv (Dataset)

Model Results Report

Business Insights Document

Visualizations (PNG files including SHAP, Segmentation, Revenue Trends, and ROC Curves)

Project Demo Video

Instructions to Run the Project:

Open Google Colab or Jupyter Notebook.

Upload the Task-1_ ML-INT-1 .ipynb file.

Upload the WA_Fn-UseC_-Telco-Customer-Churn.csv dataset to the working directory.

Run the cells sequentially from top to bottom.

Note: Ensure the shap library is installed (!pip install shap) before running the final bonus challenge cell.
